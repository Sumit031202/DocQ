# DocQ - Patient Queue Management System

DocQ is a real-time smart queuing system designed for clinics and hospitals to manage patient flow efficiently.

## Features
- **Real-time Queue Updates**: Patients see their position live.
- **Smart Wait Time**: Calculates wait time based on actual service speed.
- **Duplicate Prevention**: Prevents patients from joining multiple times.
- **Admin Dashboard**: Doctors/Front desk can Call, Skip, or Complete patients.

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Supabase (Auth, Database, Realtime)

## Setup
1. Clone the repo.
2. `npm install`
3. Setup `.env.local` with Supabase credentials.
4. `npm run dev`
